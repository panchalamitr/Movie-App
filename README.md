# Movie-App-Android-MVVM-with-Coroutine

Movie app is an open source application, created using below tools: 

- MVVM pattern 
- Live Data 
- Coroutine
- Retrofit 
- Dagger Hilt
- Data Binding
- Material Design 
- Testing with Junit4
